Instrukcja szybkiego startu Stick firmware

Sterowanie notatnika (w nawiasie - Cardputer):
Do gory - Przycisk C (ESC)
W dol - Przycisk B (Tab)
Wyjdz - Przycisk A (Enter)

Sterowanie systemu:
Zmien pozycje - Przycisk B (Tab)
Potwierdz / OK - Przycisk A (Enter)
Wyjdz / Anulus - Przycisk C (ESC)

Clock controls:
Menu aplikacji - Przycisk A (Enter)
Menu zegara - Przycisk B (Tab)
Menu blokad - Przytrzymaj przycisk B (Tab) przez 1 second
Menu blokad (W trybie atrapy) - Trzymaj Przycisk A i B (Enter i Tab) przez 1 sekunde
Szybkie akcje - Przycisk C (ESC)

W trybie atrapy wszystko oprocz menu zasilania jest zablokowane!
