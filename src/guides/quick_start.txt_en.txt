Stick firmware quick start guide

File reader controls (Cardputer):
Up - Button C (ESC)
Down - Button B (Tab)
Exit - Button A (Enter)

System controls:
Cycle - Button B (Tab)
Confirm / OK - Button A (Enter)
Exit / Cancel - Button C (ESC)

Clock controls:
Apps menu - Button A (Enter)
Clock menu - Button B (Tab)
Lock menu - Hold Button B (Tab) for 1 second
Lock menu (While in dummy mode) - Hold Button A and B (Enter and Tab) for 1 second
Quick actions - Button C (ESC)

When in dummy mode everything except power menu is locked!!!
